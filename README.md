# blockchain-example
A little example of a blockchain in python

### Subir o programa:
```
python app.py
```

### Minerar bloco
GET http://localhost:5000/mine-block HTTP/1.1

### Ver a blockchain
GET http://localhost:5000/get-blockchain HTTP/1.1

### Checar a blockchain
GET http://localhost:5000/blockchain-is-valid HTTP/1.1
